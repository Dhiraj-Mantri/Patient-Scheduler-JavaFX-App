# Patient Scheduler — JavaFX Desktop App

A desktop-based Patient Appointment Management System built with Java 21 and JavaFX 21.
Developed as a university project at Bauhaus-Universität Weimar (SE 2026).

## Features
- Browse available doctors with their specialties and time slots
- Book appointments with real-time conflict detection
- View all appointments with live search by patient ID or doctor name
- Automatic CSV persistence — appointments survive app restarts
- Clean dark-themed UI with sidebar navigation

## Tech Stack
- Java 21
- JavaFX 21 (UI framework)
- Apache Commons CSV (data persistence)
- Gradle (build tool)
- H2 (database dependency — future upgrade)

## How to Run
Make sure you have Java 21 installed, then:
git clone https://github.com/Dhiraj-Mantri/Patient-Scheduler-JavaFX-App.git
cd Patient-Scheduler-JavaFX-App
./gradlew run
